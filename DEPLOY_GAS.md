# GitHub Pages + Google Таблица + Apps Script

Самый простой вариант для менеджеров: **публичная ссылка GitHub Pages**, данные в **Google Таблице**, API через **Apps Script**. Бесплатно, без серверов и MongoDB.

---

## Схема

```
Менеджер → GitHub Pages (интерфейс)
              ↕ fetch
         Apps Script (API)
              ↕
         Google Таблица (данные)
```

---

## Шаг 1. Google Таблица + Apps Script (~5 мин)

1. Создайте [новую Google Таблицу](https://sheets.new).
2. **Расширения → Apps Script**.
3. Удалите содержимое `Code.gs` и вставьте код из файла [`gas/Code.gs`](./gas/Code.gs) этого репозитория.
4. Сохраните проект (Ctrl+S).
5. В редакторе выберите функцию **`setup`** → **Выполнить** → разрешите доступ к таблице.
   - `setup` создаёт листы `_pipeline`, `_audit`, `_snapshots_daily`, `_snapshots_deals` и ставит **триггер снапшота в 23:59 МСК** ежедневно.
   - Для первого снапшота сразу: откройте `ВАШ_URL/exec?action=snapshotNow`
6. **Развернуть → Новое развертывание**:
   - Тип: **Веб-приложение**
   - Запуск от имени: **Я**
   - Доступ: **Все** (в том числе анонимные)
7. Скопируйте **URL веб-приложения** (вида `https://script.google.com/macros/s/AKfy.../exec`).

Проверка: откройте в браузере  
`https://script.google.com/macros/s/ВАШ_ID/exec?action=health`  
→ должно вернуть `{"ok":true,...}`

---

## Шаг 2. Подключить URL к фронтенду

Откройте `js/gas-config.js` и вставьте URL:

```javascript
window.ITMEN_GAS_CONFIG = {
  url: "https://script.google.com/macros/s/ВАШ_ID/exec",
};
```

Закоммитьте и запушьте в GitHub.

---

## Шаг 3. GitHub Pages (~2 мин)

1. Репозиторий → **Settings → Pages**
2. **Source**: GitHub Actions (рекомендуется — workflow уже в репозитории)

   *или* **Deploy from branch** → `master` → `/ (root)`

3. После деплоя ссылка для менеджеров:

```
https://dantrusov10.github.io/itmen-pipeline/
```

(замените `dantrusov10` на ваш GitHub-логин, если другой)

## Готово

Отправьте менеджерам ссылку GitHub Pages. Вход не нужен — открыли, заполняют, жмут **💾 Сохранить** → данные уходят в вашу Google Таблицу (лист `_pipeline`, скрытый).

Таблицу можно открыть в любой момент для просмотра/бэкапа. Экспорт JSON — кнопка **⬇️ Экспорт JSON** в интерфейсе.

## Локальная разработка

| Режим | Как |
|---|---|
| Только UI | `ITMEN_API.enabled = false` в `api.js` → localStorage |
| Express (старый) | `npm start` на localhost |
| Через Google | в `gas-config.js` указать URL + `ITMEN_FORCE_GAS = true` в консоли |

## Частые проблемы

**«Ошибка сохранения» / CORS**  
Убедитесь, что веб-приложение развёрнуто с доступом **«Все»**, и используете URL из **Развернуть**, не из редактора.

**Данные не появляются**  
Проверьте лист `_pipeline` в таблице (Правка → Показать скрытые листы).

**После изменения Code.gs**  
Создайте **новое развертывание** (или «Управление развертываниями → Изменить → Новая версия»).
